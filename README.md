# Prueba tecnica

Instrucciones como correr proyecto localmente

clone repo

```
git clone ...
```

## Server

Una vez dentro de la carpeta que contiene el proyecto
ir a la carpeta del Server

```
cd Server
```

instalar dependencias

```
npm i
```

correr servidor

```
npm run dev
```

## Front-end

Una vez dentro de la carpeta que contiene el proyecto
ir a la carpeta del front-end

```
cd front-end
```

instalar dependencias

```
npm i
```

correr servidor

```
npm run dev
```
